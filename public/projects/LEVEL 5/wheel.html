Interactive Zoetrope Illusion

A web experiment using SVG and Tailwind CSS animation

How It Works (Concept)

This interactive element demonstrates the principle behind a Zoetrope, an early device that created the illusion of motion.

Inside the "drum", there are several images (represented here by colored SVG triangles). In a real zoetrope, each image would be a slightly different frame of an animation sequence.

When the drum spins, your eyes glimpse each image briefly. Due to a phenomenon called persistence of vision, your brain blends these rapidly changing images together, creating the perception of smooth movement.

This web version uses CSS animations (powered by Tailwind CSS utility classes and a touch of custom CSS) to rotate the container holding the SVG "slices". Click the button above to start and stop the rotation!
