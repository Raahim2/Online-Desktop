Vulnerability Disclosure Program

Program Guidelines

  • No brute force attacks - automated mass requests will result in IP blacklisting
  • No mass file uploads - uploading excessive files is not a vulnerability
  • Stay within ethical boundaries - we monitor all activities

Scope

We welcome reports on security issues including but not limited to:

  • Cross-Site Scripting (XSS)
  • SQL Injection
  • Unauthorized Database Access
  • Authentication Bypasses
  • Other Security Vulnerabilities
