About This Simulator

This interactive orrery provides a simplified visual representation of celestial bodies moving under the influence of mutual gravitational attraction. An orrery is a mechanical model of the solar system that illustrates the relative positions and motions of the planets and moons.

Use the controls to customize the simulation parameters. Adjust the Gravitational Constant (G) to see how it affects orbital speeds and stability. Change the Number of Planets or load different Preset Systems. The Time Scale allows you to speed up or slow down the simulation.

Note: This is a visual demonstration and may not represent perfectly accurate physics. The simulation logic itself is not implemented in this HTML/CSS-only prototype. The buttons provide basic interaction feedback via console logs and status updates.
